**National Tsing Hua University (NTHU)** | 指導教授：孫宏民 教授 

這是一個從零開始架構的去中心化平台，旨在解決自由職業市場中割裂、高額中介費及仲裁不透明的問題 。

#### **核心技術貢獻 (Technical Contributions)**

**一站式去中心化系統架構**：整合 **React DApp**、**Polygon PoS 智能合約**、**IPFS 分散式存儲**與 **XMTP 加密通訊**，構建完整的 Web3 生態系 。

**混合加密交付機制 (Hybrid Encryption)**：實作 **AES + ECDH** 加密技術，確保數位資產在公共 IPFS 節點上的隱私性，實現「無需信任 (Trustless)」的端到端安全交付 。

**DAO 去中心化仲裁系統**：設計基於代幣質押 (Staking) 與隨機選拔機制的 **DAO 治理模型**，由社區陪審團透過鏈上投票解決交易糾紛，取代傳統中心化客服 。

**高效率與低成本優化**：透過智能合約優化將交易手續費從傳統平台的 20% 降低至約 **$0.01 USD** 。

### 深度演算法優化：Akagi 專案

#### **[Akagi v3.0: 台灣麻將 AI 輔助與全知視角覆盤系統]**

基於開源工具進行深度重構與功能進階開發，將其演化為適用於 16 張台麻的決策引擎。

* **MVC 嚴格解耦架構**：將系統重構為 **Model-View-Controller** 模式，徹底分離數學大腦與前端渲染，提升系統擴展性與維護效率。
* **混合動力雙引擎 (Dual-Engine)**：導入 **馬可夫時間模型**，根據向聽數動態切換決策模式，精準預測預期巡目與鳴牌閾值。
* **向聽外改良引擎 (Secondary Ukeire)**：解決搭子超載時的決策盲點，透過二次極限搜索評估形狀潛力期望值 ($EV$)。
* **時空覆盤系統 (Replay System)**：實作事件溯源 (Event Sourcing) 架構，支援歷史對局的狀態還原與 AI 戰術回批分析。
